# Georgia Annual Registration — Template for P31 Labs, Inc.

**Entity type:** Nonprofit corporation  
**Jurisdiction:** Georgia  
**Filing frequency:** Annual  
**Deadline:** **April 1** each year (fixed statewide window, not based on formation anniversary)  
**Late fee:** $400 if filed after April 1  
**Estimated fee:** $40 (standard nonprofit annual registration fee)  
**Form:** **Annual Registration** filed with Georgia Secretary of State  
**Priority:** File immediately if deadline has passed — $400 late fee is preferable to administrative dissolution (July–September window)

---

## Pre-Flight Checklist

- [ ] Confirm entity is in **good standing** with Georgia SOS
- [ ] Locate your **Entity ID / Control Number** (from incorporation docs)
- [ ] Verify **registered agent** information (name, address, email)
- [ ] Confirm **principal office address** (can be physical or virtual in GA)
- [ ] Gather **director/officer list** with full legal names and addresses
- [ ] Confirm EIN: **42-1888158** is still current with IRS
- [ ] Ensure **no name/address changes** since last filing (if yes, file amendments first)

## Filing Instructions

1. **Online (recommended):**  
   Go to [https://ecorp.sos.ga.gov](https://ecorp.sos.ga.gov)  
   Log in → Annual Registrations → File Annual Registration  
   Pay fee with credit card (instant confirmation)

2. **Mail (if preferred):**  
   Download form from [https://sos.ga.gov/annual-registration](https://sos.ga.gov/annual-registration)  
   Mail check + form to:  
   > Georgia Secretary of State  
   > Corporations Division  
   > 2 MLK Jr. Dr. SE, Suite 313 West Tower  
   > Atlanta, GA 30334-1530

3. **Confirmation:** Save the **filed stamp** + payment receipt. Attach to corporate records.

## Notes

- If the entity is **inactive** or **dissolved**, you cannot file an annual registration — you must reinstate first.
- If you miss the deadline, a **late fee** applies (typically $35–$100 depending on entity type).
- Keep a copy of the filed document in your corporate records binder (digital is fine).
